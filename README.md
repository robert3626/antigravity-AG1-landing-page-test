# Creative Marketing Agency Landing Page

A dark-themed, ultra-premium landing page built with modern HTML5, Vanilla CSS, and JavaScript.

## Features
- **Modern Dark Glassmorphic Design**: Pitch black aesthetic with warm glowing amber accents.
- **Hero & Intro Section**: Clean brand overview and 3-card feature grid.
- **Key Features Showcase**: Spotlight cards (`01` and `02`) with custom 3D artwork.
- **Flexible Pricing Grid**: Starter, Professional (Featured), and Enterprise plans with an interactive Monthly/Annual billing toggle.
- **Interactive Contact Modal**: Integrated email form modal for sales inquiries.

## Visual Assets
All 3D render visual assets are stored in the `assets/` directory.

## Local Setup
To run locally, simply open `index.html` in any modern browser or serve via Python:
```bash
python -m http.server 8080
```
Then visit `http://localhost:8080`.
